Recipe for ML

  1. Given data ${x_i, y_i}_{i=1}^N$
  2. Choose decision function and loss function
    • $\hat{y}i = f{\theta}(x_i)$
    • $\ell(\hat{y}, y)\in\mathbb{R}$
  3. Define goal: $\theta^* = \argmin \ \sum\ell(f_{\theta}(x_i),y_i)$
  4. Train with SGD

Things to work on: 2 & 4

Decision Functions (One hidden layer)

  • Input: Given $x_i, \forall i$
  • Hidden (linear): $a_j = \sum_{i=0}^M\alpha_{ji}x_i, \forall j$ (bias term is folded in)
  • Hidden (sigmoid): $z_j = sigmoid(a_j) = \frac{1}{1+\mathrm{exp}(-a_j)}, \ \forall j$
  • Output (linear): $b = \sum_{j=0}^D\beta_jz_j$
  • Output (sigmoid): $y = sigmoid(b) = \frac{1}{1+\mathrm{exp}(-b)}$

NN Architectures

  • Number of hidden layers
  • Number of hidden units
  • Type of activation function (nonlinearity)
  • Form of objective function
  • Theoretically, a neural network with 1 hidden layer is a universal function approximator

    Cybenko (1989): For any continuous function g(x), there exists a 1-hidden-layer neural net $h_{\theta}(x)$ s.t. $$| h_{\theta}(x) – g(x) | < \epsilon$$ for all x, assuming sigmoid activation functions

Activation Functions

  • Sigmoid / Logistic Function $[0,1]$: $$\mathrm{sigmoid}(u)\equiv \frac{1}{1+\mathrm{exp}(-u)}$$
  • $tanh(x) \in [-1,1]$
  • $reLU(z) = max{0,z}$

Multi-Class Output

  • Input: Given $x_i, \forall i$
  • Hidden (linear): $a_j = \sum_{i=0}^M\alpha_{j,i}x_i, \forall j$ (bias term is folded in)
  • Hidden (nonlinear): $z_j = \sigma(a_j), \ \forall j$
  • Output (linear): $b_k = \sum_{k=0}^D\beta_{k,j}z_j, \ \forall k$
  • Output (nonlinear): $\hat{y}k = \mathrm{softmax}(b_k) = \frac{\mathrm{exp}(b_k)}{\sum{\ell=1}^K\mathrm{exp}(b_\ell)}$
  • Loss (cross-entropy): $$J = -\sum_{k=1}^Ky_k\mathrm{log}(\hat{y}_k)$$
    • $\hat{y}_k$ is the probability of predicting label $y_k$ over the K classes
