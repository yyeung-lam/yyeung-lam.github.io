Sliding Window

Sliding Window

Reusing common elements in a sliding window.

In some problems, the size of the sliding window is not fixed.

Smalllest subarray with given sum s (easy)

The size of sliding window is not fixed.

  1. Add up elements until the sum is greater or equal to s
  2. These element will constitute our first sliding window
  3. Keep adding elements, and try to shrink the size of sliding window until the sum is smaller than s. This shrinking will happen in multiple steps. In each step:
    1. Update smallest window length
    2. Subtract the first element of the window from the running sum to shrink the sliding window
using namespace std;

#include <iostream>
#include <limits>
#include <vector>

class MinSizeSubArraySum &#123;
 public:
  static int findMinSubArray(int S, const vector<int>& arr) &#123;
    int start = 0, windowSum = 0; 
    int minL = numeric_limits<int>::max();
    for (int end = 0; end < arr.size(); end++) &#123;
      windowSum += arr[end];
      while (windowSum >= S) &#123;
        minL = min(minL, end - start + 1);
        windowSum -= arr[start];
        start++;
      &#125;
    &#125;
    return minL == numeric_limits<int>::max() ? 0 : minL;
  &#125;

Longest Substring with Same Letters after Replacement (hard)

Idea: Keep track of maximum frequency in the sliding window. Then the number of replacement = end - start + 1 - maximum frequency

Note: The max_freq need not be updated to a lesser value when shrinking the window size. Because we only care if the size can enlarge.

class Solution &#123;
public:
    int characterReplacement(string s, int k) &#123;
        int longest = 0, max_freq = 0;
        int start = 0;
        unordered_map<char, int> freq_map;
        for (int end = 0; end < s.length(); end++) &#123;
            freq_map[s[end]]++;
            max_freq = max(max_freq, freq_map[s[end]]);
            if (end - start + 1 - max_freq > k) &#123;
                freq_map[s[start]]--;
                start++;
            &#125;
            longest = max(longest, end - start + 1);
        &#125;
        return longest;
    &#125;

    class Solution:
    def subarraysWithKDistinct(self, A: List[int], K: int) -> int:
        def atMostK(M):
            result = 0
            counter = collections.Counter()
            left = 0
            for right, a in enumerate(A):
                counter[a] += 1
                while len(counter) > M:
                    counter[A[left]] -= 1
                    if counter[A[left]] == 0:
                        del counter[A[left]]
                    left += 1
                result += right-left+1 # given right endpoint, how many continguous subarray?
