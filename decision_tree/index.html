Decision Tree

Learning Objectives

  • Implement decision tree training and prediction
  • Use effective splitting criteria for decision tree and be able to define entropy, conditional entropy, and mutual information/gini gain
  • Explain the difference between generalization and memorization [CIML]
  • Describe the inductive bias of a decision tree
  • Formalize a learning problem by identifying the input space, output space, hypothesis space, and target function
  • Explain the difference between training error and training error
  • Judge whether a decision tree is “overfitting” or “underfitting”
  • Implementing a pruning or early stopping method to combat overfitting in Decision Tree Learning

(Shallow) Decision Tree

  • Inductive Bias: Decisions can be made by only looking at small number of features
  • Inductive Bias source of error: learning algorithm cannot cope with data

Splitting criteria

  • Gini Impurity: Expected Misclassification Rate

    • Gini Impurity of dataset D: $G(Y;D) = \sum_{k=1}^KP(Y=k)\cdot P(Y\neq k) = 1-\sum_{k=1}^KP(Y=k)^2$ if binary
    • Gini Gain that splits on X: $$G(Y,X;D) = G(Y;D) - \sum_{x=1}^nP(X=x)G(Y;D_{X=x})$$
    • Gini Gain: How much pure the dataset becomes after split

  • Mutual Information

    • Entropy: the expected # of bits to code a random draw from ., DT tries to reduce entropy of the random variable
      $$H(Y) = -\sum_{i=1}^nP(Y=i)\mathrm{log}_2P(Y=i)$$
    • Specific Conditional Entropy:
      $$H(Y|X=x) = -\sum_{i=1}^nP(Y=i|X=x)\mathrm{log}_2P(Y=i|X=x)$$
    • Conditional Entropy: expected value of specific conditional entropy
      $$H(Y|X) = \sum_{x\in range(X)}P(X=x)H(Y|X=x)$$
    • Mutual Information: If we know X, how much does it reduce the uncertainty about Y?
      $$I(X,Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

Underfitting

  • Too simple, unable to capture trends in data, exhibits too much bias
  • Examples: majority-vote classifier

Overfitting

  • Too complex, fitting noise in data or fluctuations inherent in sample, does not have enough bias
  • Examples: Memorizer
