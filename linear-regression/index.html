Some Thoughts about Linear Regression

Some thoughts about linear regression

Starter Goal: Try to predict some random variable Y

Mean squared error
$$MSE([Y-m]^2)$$
derives the optimal prediction $\mu = E[Y]$.

Predict Y with regressor X

$$MSE([Y-f(X)]^2)$$
derives the optimal function $f(x) = E[Y|X=x]$

Suppose the function is linear

Again by minimizing $MSE(\beta) = E([Y-\beta_0-\beta X]^2)$, we derive optimal $\beta$:
$$\beta = Var(X)^{-1}Cov(X,Y)$$
$$\beta_0 = E[Y] - \beta E[X]$$

  • Note: $\beta$ depends on marginal distribution of $X$. If that shifts, $\beta$ will shift unless the regression function is linear.

Prediciton and error

  • Covariance identity: $Cov(X+Z, Y) = Cov(X,Y)+Cov(Z,Y)$
  • prediction error is uncorrelated with regressor:

$$Cov(Y-\beta X, X) = Cov(Y,X)-Cov([Var(X)^{-1}Cov(X,Y)]X,X)$$

$$ = Cov(Y,X) - Var(X)^{-1}Cov(X,Y)Var(X)$$

  • Expected prediction error $E[Y-X\beta] = 0$
  • Although in general $E[Y-X\beta|X=x]\neq 0$
  • And $Var[Y-X\beta|X=x_1] \neq Var[Y-X\beta|X=x_2]$

Estimating optimal Linear Predictor

To actually estimate $\beta$, we need some assumptions:

  • $(X_i, Y_i)$ are independent of each other
  • $Cov(X_i, Y_i)$ unchanging
  • By CLT, $\frac{1}{n}X^TY \rightarrow Cov[X, Y]$, $\frac{1}{n}X^TX\rightarrow Var(X)$

So that we have consistent estimator $\hat{\beta} = (X^TX)^{-1}X^TY \rightarrow \beta$

If more assumptions are made: $Y = X\beta + \epsilon$,

$$\hat{\beta} = (X^TX)^{-1}X^T(X\beta + \epsilon) = \beta + (X^TX)^{-1}X^T\epsilon$$
So we derive:

  • $E[\hat{\beta}|X= x] = \beta + (X^TX)^{-1}X^TE[\epsilon] = \beta$
  • $Var(\hat{\beta}|X=x) = Var((X^TX)^{-1}X^T\epsilon|X=x) = (X^TX)^{-1}X^T\Sigma(x)X(X^TX)^{-1}$

where $Var(\epsilon|X=x) = \Sigma(x)$. If errors are uncorrelated and equal diagonal variance,
$\Sigma(x) = \sigma^2I$, $Var(\hat{\beta}|X=x) = \sigma^2(x^Tx)^{-1} = \frac{\sigma^2}{n}(x^Tx)^{-1}$

This tells us that variance of estimated linear regression coefficients goes down as $n$ increases, goes up as $\sigma^2$ increases,
and goes down when regressors have more sample variance, less correlated with each others
