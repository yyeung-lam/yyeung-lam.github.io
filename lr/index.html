Logistic Regression

Probabilistic Learning

Assume our output is sampled from a conditional probability distribution:

  • $\mathbf{x}^{(i)}\sim p(\cdot)$
  • $y^{(i)}\sim p(\cdot|\mathbf{x}^{(i)})$

Goal: Learn a probability distribution $p(y|\mathbf{x})$ that best approximates the real $p$.

Maximum Likelihood Estimation

$$\theta^{MLE} = \underset{\theta\in \Theta}\mathrm{argmax}\prod_{i=1}^Np(\mathbf{x}^{(i)}|\theta)$$

  • MLE tries to allocate as much probability mass as possible to observed data, at the expense of things we have not seen.

Binary Logistic Regression

Define a linear classifier
$$\mathcal{D} = {\mathbf{x}^{(i)}, y^{(i)}}_{i=1}^N, \text{where } \mathbf{x}\in\mathbb{R}^M \text{and } y\in{0, 1}$$
Define an objective function
Use a differentiable function: $logistic(u) = \frac{1}{1+e^{-u}}$

  • $$p_{\theta}(y=1|\mathbf{x}^{(i)}) = \frac{1}{1+\mathrm{exp}(-\theta^T\mathbf{x}^{(i)})} = \sigma(\theta^T\mathbf{x}^{(i)})$$
  • $$p_{\theta}(y=0|\mathbf{x}^{(i)}) = 1 - \sigma(\theta^T\mathbf{x}^{(i)})$$

Succinctly we can write: $$J(\theta) = -\sum_{i=1}^N\mathrm{log}\bigg(\sigma(\theta^T\mathbf{x}^{(i)})^{y^{(i)}}(1-\sigma(\theta^T\mathbf{x}^{(i)})^{1-y^{(i)}})\bigg)$$

Optimize it with GD/SGD to learn parameters
$$\theta^* = \underset{\theta\in\Theta}\mathrm{argmin} \ J(\theta)$$
where we minimize the negative log conditional likelihood (equivalent to maximizing Conditional Likelihood Estimation):
$$J(\theta) = -\mathrm{log}\prod_{i=1}^Np_{\theta}(y^{(i)}|\mathbf{x}^{(i)}) = \sum_{i=1}^NJ^{(i)}(\theta)$$

Why NLL?

  • Some likelihood can’t be maximized (MLE parameters for logistic regression)
  • Don’t have joint model $p(x,y)$
  • Works well on linear regression

Logistic Regression MCLE

Partial derivative of example $i$ w.r.t $k^{th}$ feature:
$$J^{(i)}(\theta) =-p_{\theta}(y^{(i)}|\mathbf{x}^{(i)}) = -\bigg(y^{(i)}\mathrm{log}(\sigma(\theta^T\mathbf{x}^{(i)}))+(1-y^{(i)})\mathrm{log}(1-\sigma(\theta^T\mathbf{x}^{(i)}))\bigg)$$

$$\frac{\partial}{\partial \theta_k}J^{(i)}(\theta) = -\frac{y^{(i)}}{\sigma(\theta^T\mathbf{x}^{(i)})}(\sigma(\theta^T\mathbf{x}^{(i)})(1-\sigma(\theta^T\mathbf{x}^{(i)}))x^{(i)}_k)$$

$$+\frac{(1-y^{(i)})}{\sigma(1-\theta^T\mathbf{x}^{(i)})}(\sigma(\theta^T\mathbf{x}^{(i)})(1-\sigma(\theta^T\mathbf{x}^{(i)})x^{(i)}_k)$$

$$=(-y^{(i)}(1-\sigma(\theta^T\mathbf{x}^{(i)})+(1-y^{(i)})\sigma(\theta^T\mathbf{x}^{(i)}))x^{(i)}_k$$

$$=(\sigma(\theta^T\mathbf{x}^{(i)})-y^{(i)})x^{(i)}_k$$

SGD:

  1. randomly pick an example: $i\sim \text{shuffle}(1,…,N)$
  2. compute the gradient of the log-likelihood for that example: $g \leftarrow J^{(i)}(\theta) = -\mathrm{log}p_{\theta}(y^{(i)}|\mathbf{x}^{(i)})$
  3. update all the parameters using that gradient: $\theta_k\leftarrow \theta_k - \lambda g$

Predict the class with highest probability under the model
$$\hat{y} = \underset{y\in{0,1}}\mathrm{argmax}\ p_{\theta}(y|\mathbf{x}) = \mathbb{1}_{(p \geq 0.5)}$$

Multinomial Logistic Regression

for $k$ in $1,…,K$,
$$\frac{\partial J_i(\Theta)}{\partial \theta_k} = -(T_{i,j}-p(k|\mathbf{x}^{(i)},\Theta))\mathbf{x}^{(i)}$$
for $k$ in $1,…,K$,
$$\theta_k \leftarrow \theta_k - \gamma \frac{\partial J_i(\Theta)}{\partial \theta_k}$$
