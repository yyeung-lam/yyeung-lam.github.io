Recommender System

Recommender System

  • Setup: Items (products), Users (watchers), Feedback (ratings)
  • Key Assumptions:
    • Represent rating as user/item (sparse) matrix
  • Two Types of Recommender System
    • Content Filtering: Side information (cons), easy to add new item (pros)
    • Collaborative Filtering: No access to side information (pros) just the matrix, does not work on new item (adding empty column to matrix, cons)
      • Neighborhood method: find neighbors, recommend same items
      • Latent factoring method: recommend based on proximity in low dimensional space (Matrix Factorization)

Low-Rank Factorization

  1. rank $K << \min(M,N)$,

Given: $M\times N$ matrix, $R$ of ($M$ users, $N$ items for rating)
Decompose into: $U\times V^T$, where $U$ is $M\times K$ and $V$ is $N\times K$
Idea: Rating entry = dot product of user feature vector and item vector

  1. rank $\ell > K$,

Decompose into: $R \approx U\times V^T$

Approximate Error: $E = R - U\times V^T$, $MSE = |E|^2_2$ Frobenius Norm

Unconstrained Matrix Factorization (UMF)

Fully Observed $R$
$$\hat{U}, \hat{V} = \argmin\frac{1}{2}|R-U\times V^T|^2_2$$
Partially Observed $R$

  • $r_{ij} = R[i,j], \vec u_i = U[i,], \vec v_j = V[,j], Z = {r_{ij} \text{which are observed}}$
    $$J(U,V) = \frac{1}{2}\sum_{(i,j)\in Z}(r_{i,j}-\vec u_i^T\vec v_j)^2 = \frac{1}{2}\sum_{(i,j)\in Z}J_{i,j}(U,V)$$

Alternating Least Squares (ALS) for UMF

  • Just like block coordinate descent

Singular Value Decomposition for Collaborative Filtering

Theorem: If $R$ fully observed and no regularization, the optimal UVT from SVD equals the optimal UVT from UMF
